\subsection{Detailed Description}
Class to hold information about the presentation. 

This class hold the information about the current status of the presentation. It holds the U\+R\+Ls to the slides, the number of total slides and the current slide, as well as an identifier. It is implemented thread safe to enable access from multiple threads.
